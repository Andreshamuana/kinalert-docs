🗑️ Suppression du compte

Kin Alerte

🔒 Vos données, votre contrôle

Chez Kin Alerte, nous respectons votre vie privée et vous donnons la possibilité de supprimer définitivement votre compte ainsi que toutes les données personnelles qui y sont associées.

📋 Comment supprimer votre compte ?

Vous pouvez demander la suppression de votre compte directement depuis l'application :

  1. Ouvrez l'application Kin Alerte.
  2. Connectez-vous à votre compte.
  3. Accédez au menu Profil.
  4. Sélectionnez Supprimer mon compte.
  5. Confirmez votre demande.
La suppression du compte est irréversible. Toutes vos données seront définitivement effacées.

🧹 Données supprimées

Lorsque votre demande est traitée, les informations suivantes sont supprimées :

  • Votre compte utilisateur
  • Votre numéro de téléphone
  • Votre adresse e-mail (si renseignée)
  • Votre photo de profil
  • Vos informations personnelles
  • Vos commentaires
  • Vos mentions « J'aime »
  • Vos fichiers audio
  • Vos photos
  • Vos vidéos
  • Toutes les autres données associées à votre compte

📁 Données pouvant être conservées

Certaines informations techniques ou journaux de sécurité peuvent être conservées temporairement afin de :

  • respecter les obligations légales ;
  • prévenir les fraudes ;
  • assurer la sécurité de la plateforme.

Ces données sont automatiquement supprimées dès que leur conservation n'est plus nécessaire.
